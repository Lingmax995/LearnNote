### 常用按键

#### ** 视图模式切换**​

| 快捷键            | 功能                               |     |
| -------------- | -------------------------------- | --- |
| ​**Host + F**​ | 全屏模式 ↔ 窗口模式切换（需松开 Host 键再按第二次退出） |     |
| ​**Host + L**​ | 无缝模式（Guest 应用窗口与主机桌面混合显示）        |     |
| ​**Host + C**​ | 缩放模式（内容随窗口大小自适应缩放）               |     |
| ​**Host + A**​ | 自动调整虚拟机窗口大小                      |     |
| ​**Host + G**​ | 自动调整虚拟机显示分辨率（适配 Guest 系统）        |     |
#### **虚拟机控制**​

| 快捷键              | 功能                                     |
| ---------------- | -------------------------------------- |
| ​**Host + Del**​ | 向 Guest 发送 `Ctrl+Alt+Del` 信号（如调出任务管理器） |
| ​**Host + R**​   | 重启虚拟机                                  |
| ​**Host + H**​   | 正常关闭虚拟机（安全关机）                          |
| ​**Host + P**​   | 暂停/恢复虚拟机运行                             |
| ​**Host + Q**​   | 强制退出虚拟机（关闭窗口）                          |
#### **辅助功能**​

| 快捷键                | 功能                                  |
| ------------------ | ----------------------------------- |
| ​**Host + Home**​  | 显示隐藏的菜单栏（如误操作导致菜单消失时使用）             |
| ​**Host + I**​     | 禁止/允许自动捕获鼠标（避免鼠标被虚拟机锁定）             |
| ​**Host + D**​     | 安装/升级 VirtualBox 增强功能（需 Guest 系统支持） |
| **“Win+空格”可切换中英文** |                                     |
